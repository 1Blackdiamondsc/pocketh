# pocketh
A pocket knife for auditing smart contracts.

## Installation:
```
npm install pocketh
```

## Usage:
```
pocketh <command> [options]
```

## Documentation:
```
pocketh --help
```
or
```
pocketh <command> --help
```

## Network cofiguration:
Commands access networks by name. To define a name, edit `./config.json`.
